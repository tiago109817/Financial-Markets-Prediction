# Financial-Markets-Prediction
BSc Thesis: Comparing the efficiency of ML models, ARIMA models, and the random walk at forecasting the evolution of financial markets 

main.py - main file, used for execution

data.py - read and prepare data from excel

arima.py - create arima models

mlearning.py - create Gradient Boosting and RNN models

visual.py - visualize all the data and results
